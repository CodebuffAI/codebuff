# Strange Loop Agent

A self-modifying agent that maintains its own context and goals.

## Setup

1. Copy `.env.example` to `.env`:
```bash
cp .env.example .env
```

2. Add your OpenAI API key to `.env`:
```
OPENAI_API_KEY=sk-your-api-key-here
```

3. Install dependencies:
```bash
bun install
```

4. Run the agent:
```bash
bun start
```
