# Overview

Customize AI agents through **Agent Templates** - JSON files that define agent behavior.

## Why Multi-Agent Systems Work Better

Codebuff uses specialized agents that collaborate instead of one agent doing everything:

- **Code Generation** - Write clean, functional code
- **Review** - Catch bugs, security issues, style violations
- **Research** - Find documentation and examples
- **Planning** - Break down complex requirements
- **File Discovery** - Navigate large codebases

**Example workflow:** "refactor this authentication system"

1. **File Picker** finds auth-related files
2. **Research** looks up best practices
3. **Planning** creates step-by-step plan
4. **Base** implements changes
5. **Reviewer** checks for security issues

Agents spawn other agents, share tools, and pass context between tasks.

## Common Workflows

**Single-Agent Tasks:**
- Code review, documentation, research

**Multi-Agent Workflows:**
```
Feature: Planning → File Picker → Base → Reviewer
Debugging: Thinker → Research → Base → Reviewer  
Refactoring: File Picker → Planning → Base → Reviewer
```

**Team Sharing:**
```bash
git add .agents/
git commit -m "Add team coding standards"
```

**Role-Specific Examples:**
- Frontend: React component reviewer
- Backend: API security reviewer
- DevOps: Infrastructure deployment agent
