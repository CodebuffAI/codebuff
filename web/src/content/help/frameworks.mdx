# Language & Framework Support

## Supported Languages

Codebuff provides intelligent assistance across many programming languages:

### Web Technologies
- JavaScript/TypeScript
- HTML/CSS
- React, Vue, Angular
- Node.js, Express

### Systems Programming
- Python
- Java
- C/C++
- Rust
- Go

### Mobile Development
- Swift
- Kotlin
- React Native
- Flutter

## Framework-Specific Features

Codebuff understands framework-specific patterns and best practices:

### React Applications
- Component lifecycle management
- State and props handling
- Hook usage patterns
- Performance optimization

### Express/Node.js
- Route management
- Middleware implementation
- Error handling
- API design

### Database Integration
- Query optimization
- Schema management
- Migration handling
- ORM integration

## Configuration Files

Codebuff can help with various configuration files:

- package.json
- tsconfig.json
- webpack/vite config
- docker-compose.yml
- CI/CD configurations

## Best Practices

- Let Codebuff detect your project's framework
- Use framework-specific commands when available
- Leverage built-in testing capabilities
- Follow framework conventions in your requests
