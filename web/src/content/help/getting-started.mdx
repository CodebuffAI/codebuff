# Getting Started

Welcome! Let's get you set up with Codebuff, the AI coding agent.

## Installation

Install Codebuff globally using npm:

```bash
  npm install -g codebuff
```

## Basic Usage

Start using Codebuff in your project directory:

```bash
  codebuff
```

Or, start a new project with one of our [starter templates](https://github.com/CodebuffAI/codebuff-community/tree/main/starter-templates):

```bash
  codebuff --create nextjs project-name
```

That's it! Codebuff is now ready to help you with your coding tasks.

## Troubleshooting Install

<ul className="list-disc pl-6 space-y-2 mt-2 text-muted-foreground">
  <li>
    Make sure you have <a href="https://nodejs.org/en/download" target="_blank" rel="noopener noreferrer">Node version 18+</a> installed.
    <ul className="list-circle pl-6">
      <li>Check your version by running <code className="text-sm">node --version</code></li>
    </ul>
  </li>
  <li>
    If you get a permissions error during install, use <code className="text-sm">sudo npm install -g codebuff</code> on Mac/Linux, or run a command prompt as an administrator on Windows.
  </li>
  <li>
    If that still fails, you should <a href="https://nodejs.org/en/download" target="_blank" rel="noopener noreferrer">reinstall Node</a>.
  </li>
</ul>
