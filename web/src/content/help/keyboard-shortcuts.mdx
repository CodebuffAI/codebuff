# Keyboard Shortcuts

Master these keyboard shortcuts to boost your productivity with Codebuff.

## Navigation

- `Ctrl/Cmd + U`: Undo last change
- `Ctrl/Cmd + R`: Redo change
- `Esc`: Cancel current operation
- `Up/Down`: Navigate through command history

## File Operations

- `Ctrl/Cmd + S`: Save current changes
- `Ctrl/Cmd + D`: View diff of changes
