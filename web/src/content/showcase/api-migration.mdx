# API Migration Success

See how a team used Codebuff to migrate their REST API to GraphQL.

## Challenge

The team needed to migrate 50+ endpoints while maintaining backwards compatibility.

## Solution

Codebuff helped:
- Analyze existing endpoints
- Generate GraphQL schemas
- Create resolvers
- Maintain test coverage
